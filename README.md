## 평가기준 8 · 파이프라인 설계 순서

```
[기획] → [스토리보드] → [이미지 생성] → [영상 생성] → [오디오] → [후반편집] → [최종 검수]
  ↓           ↓               ↓               ↓            ↓           ↓
브랜드    씬별 구성       GPT Image 2      Sora 2      기삽입     자막·CTA    배포 준비
정의      타임코드        키프레임 3장     10초 영상   오디오     하드코딩
```

### 설계 근거
1. **기획 선행:** 메시지·타겟·톤 확정 없이 생성 시 방향성 분산 위험
2. **스토리보드 → 이미지 순서:** 씬 구조 확정 후 키프레임 생성으로 일관성 확보
3. **이미지 → 영상 순서:** 키프레임을 레퍼런스로 Sora 2에 제공, 스타일 연속성 유지
4. **오디오 후처리:** 영상 완성 후 오디오 싱크 맞춤이 역순보다 효율적
5. **자막 분리:** AI 도구의 한국어 자막 정확도 한계 → 후반 편집으로 품질 보장

```
